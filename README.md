# 📝 To-Do App

Aplicação web de lista de tarefas desenvolvida com **HTML, CSS e JavaScript puro**, focada em organização, produtividade e boas práticas de desenvolvimento front-end.

---

## 🚀 Sobre o projeto

O **To-Do App** é uma aplicação simples e funcional que permite ao usuário gerenciar suas tarefas do dia a dia de forma prática e intuitiva.

O projeto foi desenvolvido com o objetivo de fortalecer conhecimentos em JavaScript puro (Vanilla JS), especialmente na manipulação do DOM, organização de código e criação de interfaces modernas sem o uso de frameworks.

---

## 🎯 Objetivo

Este projeto tem como objetivo:

- Praticar JavaScript puro (sem frameworks)
- Trabalhar com manipulação do DOM
- Implementar um sistema completo de tarefas (CRUD)
- Aplicar boas práticas de organização e estrutura de código
- Criar um projeto real para portfólio

---

## ⚙️ Funcionalidades

- ✅ Adicionar tarefas  
- ✔️ Marcar tarefas como concluídas  
- ✏️ Editar tarefas  
- 🗑️ Excluir tarefas  
- 🎯 Filtrar tarefas por prioridade (Alta, Média, Baixa)  
- 🔍 Buscar tarefas em tempo real  
- 📊 Contador de tarefas (total e concluídas)  
- 🌙☀️ Alternar entre modo Dark e Light  
- 💾 Persistência de dados com LocalStorage  
- 📱 Interface responsiva (mobile e desktop)  
- ✨ Animações suaves  

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3** (Flexbox + Responsividade)
- **JavaScript (ES6+)**
- **LocalStorage**
